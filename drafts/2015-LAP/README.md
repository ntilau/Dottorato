# Finite Element Software (FES)

A Matlab/C++ framework for the development of ad-hoc simulation software aimed at rapid implementation of novel numerical methods based on the weighted residuals mathematical approach. Able to interact with geometries to generate meshes, one can straightforwardly implement numerical assembly and solve code to achieve a fast and robust simulation tool.

This book shows that a few notions of mathematics and physics, and some coding skills, enable the development of powerful and efficient numerical tools relying on freely available libraries.

Published as: *Contributions to the Art of Finite Element Analysis in Electromagnetics*,
LAP Lambert Academic Publishing, 2015, ISBN 978-3-659-86451-3.
